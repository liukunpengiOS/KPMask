# KPMaskView

Easy bulid a MaskView

<img src="./Screenshot.gif" width="200" alt="Screenshot" />

安装
---

使用[CocoaPods](http://cocoapods.org)安装

    pod "KPMaskView"
    
调用
---

```
呈现
[[KPMaskView shareMask] configMaskAboveView:"你的View"];
隐藏
[[KPMask shareMask] hidden];

```
